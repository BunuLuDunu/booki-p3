# <p align="center">Booki</p>
  
Ce projet est le deuxième projet de ma formation Développeuse WordPress chez OpenClassRooms. J'étais chargée d'intégrer l'interface d'un site internet qui permette aux usagers de trouver des hébergements et des activités dans la ville de leur choix, avec du code HTML et CSS.

## 🧐 Compétences évaluées   
- Intégrer du contenu conformément à une maquette avec HTML et CSS

## 🛠️ Tech Stack
- [HTML](https://developer.mozilla.org/fr/docs/Web/HTML)
- [CSS](https://developer.mozilla.org/fr/docs/Web/CSS)

## 🛠️ Installation  
```bash
git clone https://github.com/BunuLuDunu/booki-p3.git
npm install
```
